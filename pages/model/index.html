哥德尔不完备性定理

世界架构

八云哲学体系的世界架构基于哥德尔不完备性定理分割有序（因果）世界与无序世界。而有序世界与无序世界的总和通过称为void的名词所包裹。无序世界与void在八云哲学体系的世界架构中是存在的，但是通过数学工具只能探索有序世界中的知识。

有序世界的所有构成形式都可以被抽象为“物质及其相互作用”，每一个有序世界都可以由一个自洽的数学模型来表述（例如 model alpha）。有序世界之间的边界是相互作用——任意两个有序世界一旦建立任意形式的相互作用，那么这两个有序世界应当可以使用统一的数学模型表述，意即这两个有序世界可以被认为是一个更广泛的有序世界的一部分。

主体是 model alpha 级别实例的一种特殊物质形态。

哥德尔理论层提供有序世界向void演化的可能性（回归演化），这个层次的理论和普遍价值度量密切相关。

  • Model alpha

    有序世界的一个模型，基于物质及其相互作用——有人喜欢形式化，因此我们就形式化了。

  • 哥德尔不完全性定理

    哥德尔不完全性定理为 Model alpha 提供了更外层的包裹结构。至此，体系的世界架构雏形基本成型了。

  • 世界架构v1.0

    1905年3月量子论发布，两个月后狭义相对论提出；2016年3月，八云哲学体系的世界架构基本完成。

  • SpiritX

    哲学体系开始尝试指导工程实践的探索——广义的SpiritX指代一种Web前端解决方案。当前阶段主要包括一个以组件为中心的JavaScript框架SpiritXJS，以及基于该框架的基础组件库。

  • 普遍价值度量

    当我们基于八云哲学体系的世界观去看待世界时，一切过程都被抽象为状态变化序列。如果系统具有统一的价值度量标准，那么所有可能性都将朝一个方向演化；而如果系统没有统一的价值度量标准，那么任何状态变化序列都是不可比较的，选择就失去了意义。这是体系当前阶段所关注的主要问题。
